2. Page de Connexion — /auth

[CAPTURE D'ÉCRAN — Page de connexion]
  • Switch Radix Enseignant / Étudiant
  • React Hook Form + validation Zod

3. Création de Quiz — /teacher/quiz/new

[CAPTURE D'ÉCRAN — Formulaire de création]
  • Code d'accès → Affectation nominative des étudiants
  • Durée totale (minutes), Date d'expiration
  • Système de notation : Standard / Canadien (RG-11)
  • Génération en masse de N questions vides
  • RG-12 : validation bloque la publication si questions incomplètes
  • Modale d'affectation des étudiants (Quiz_Assignment)

4. Espace Étudiant — /student

[CAPTURE D'ÉCRAN — Tableau de bord étudiant]
  • Saisie de code d'accès → Cartes des quiz assignés (RG-05)
  • L'étudiant sélectionne directement un quiz depuis son tableau de bord

5. Passage du Quiz — /student/quiz/[id]

[CAPTURE D'ÉCRAN — Interface de passage]
  • Timer (compte à rebours) — RG-09 clôture automatique
  • Questions une par une — RG-08 pas de bouton Précédent
  • Sauvegarde à la volée (Student_Response)
  • RG-07 : reprise à la dernière question non validée

6. Tableau de Bord Enseignant — /teacher

[CAPTURE D'ÉCRAN — Dashboard enseignant]
  • Nombre d'étudiants affectés par quiz (plus de code)
  • Quiz Terminé/Expiré en grayscale (RG-10)
